For Loops Lesson

  1. Websites are styled with a stylesheet language called CSS (Cascading Style Sheets).
  2. If .html files tell the browser what "stuff" to show, .css files tell the browser how that stuff should look.
  3. CSS files end with .css and are usually linked to in an .html file with a stylesheet <link> tag inside the file's <head> tag .
  4. CSS can control:
    • The layout of the page (note that this page has a right and a left section - the lesson without styles didn't.)
    • The color of everything - the text, background, buttons, etc.
    • Some animations.
  5. CSS is called "cascading" stylesheets because it applies rules to elements AND all of the "child" elements within them. For example, if we declare a font for our <body> tag, this font will be used for every font on the page (every element within the body tag) unless we tell CSS to do otherwise.
  6. Similarly, if we declare styles for our <button> tag, all of the buttons on the page will have this style applied to them.
  7. But, what if we DON'T WANT all of our buttons to look the same?
  8. If an element is unique (i.e., there is just ONE of them), we can add an id attribute to its HTML tag. See Slide 2 above.
  9. CSS declarations reference the id attribute with # followed by the id attribute name.
  10. If an element is NOT unique (i.e., there is just MORE THAN ONE of them), we can add an class attribute to its HTML tag.
  11. CSS declarations reference class attributes with . followed by the class attribute name.
  12. Class and Id attributes can be named anything we like, but they cannot start with a number.
